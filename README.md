# WebSocket的demo专用仓库

## 安装依赖

需要你提前安装了nodejs

``` npm install ```

## demo目录结构

~~~
├── demo1        # 使用 `wss://echo.websocket.org`
|                
├── demo2        # 使用 `nodejs-websocket` 搭建服务端
| 
├── demo3        # 使用 `nodejs-websocket` 做一个聊天室雏形
| 
└── demo4        # 使用 `socket.io` 重构聊天室
~~~

## 尝试步骤

- 命令行进入到相关demo目录

- ```node WebSocket```

- 打开html文件进行尝试吧

> 聊天室同时打开多个浏览器标签就可以try一下
